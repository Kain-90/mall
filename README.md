# mall
> 以学习golang、gin、gorm、go-zero等为目的。

## 项目结构
```
├── api           // 接口层
├── cmd           // 项目启动cmd
├── domain        // 领域层
├── global        // 全局变量
├── initialize    // 初始化
├── model         // 模型层
├—— router        // 路由层
├── main.go       // 项目入口
├── settings.yaml // 配置文件
```

## 主要依赖
- gin
- gorm
- go-zero
- viper
- zap
- cobra
